# Log Classification Tool using Machine Learning

## Description
This project is a machine learning-based tool that classifies system logs into:
- INFO
- WARNING
- ERROR

The system supports:
- Paste log text
- Upload log file
- Real-time classification via Streamlit UI

## Technologies
- Python
- Scikit-learn
- Streamlit
- TF-IDF
- Naive Bayes

## How to run
```bash
pip install -r requirements.txt
python train_model.py
streamlit run app.py
